## Solution
要进行特殊情况处理
- str为null
- str为空串
- str只有符号
- str中包含字母
- str转int上下溢出

```java
public class Solution {
    public int StrToInt(String str) {
        if (str == null || str.length() == 0) {
            return 0;
        }
        
        int index = 0;
        int flag = 1;
        if (str.charAt(0) == '+') {
            index++;
        } else if (str.charAt(0) == '-') {
            flag = -1;
            index++;
        }
        if (index == 1 && str.length() == 1) {
            return 0;
        }
        
        long res = 0;
        for (int i = index; i < str.length(); i++) {
            char temp = str.charAt(i);
            if (temp >= '0' && temp <= '9') {
                res = res * 10 + temp - '0';
                if (flag == 1 && res > Integer.MAX_VALUE) return 0; //上溢出
                if (flag == -1 && res > -(double)Integer.MIN_VALUE) return 0; //下溢出
            } else {
                return 0;
            }
        }
        return (int)res * flag;
    }
}
```
